# Smart-IoT-System-for-Real-Time-Fruit-Vegetable-Freshness-Detection
Smart IoT system to monitor fruit &amp; veggie freshness using real-time temperature and humidity. Alerts via LCD, LED, and buzzer show “Stay Alert” or “Relax Enjoy.” Built with NodeMCU ESP8266, DHT11, 16×2 LCD (I2C), LED, and buzzer. Helps reduce food waste by preventing early spoilage with intuitive feedback.
